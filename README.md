# 🚀 ESP32 RGB TFT Controller (ILI9341) – No LEDs

A WiFi-controlled RGB display system using an **ESP32** and **ILI9341 TFT display**.  
Control vibrant colors in real-time from your **phone or PC** via a web browser.

---

## ✨ Features

- 📶 WiFi-based control (no app required)
- 🌐 Responsive web UI (mobile + desktop)
- 🎨 Real-time color switching
- 🔁 Auto color cycling mode
- ⚡ Smooth color transitions
- 💡 Maximum brightness output
- 📺 Full-screen RGB display

---

## 🛠️ Hardware Required

- ESP32 Development Board  
- ILI9341 2.8" TFT Display  
- Jumper Wires  

---

## 🔌 Pin Configuration

| TFT Pin | ESP32 Pin |
|--------|----------|
| VCC    | 3.3V     |
| GND    | GND      |
| CS     | GPIO 15  |
| RESET  | GPIO 4   |
| DC     | GPIO 2   |
| MOSI   | GPIO 23  |
| SCK    | GPIO 18  |
| MISO   | GPIO 19  |
| BL     | GPIO 5   |

---

## 📦 Software & Libraries

- Arduino Framework (PlatformIO / Arduino IDE)
- Adafruit GFX Library
- Adafruit ILI9341 Library
- WiFi (ESP32 built-in)
